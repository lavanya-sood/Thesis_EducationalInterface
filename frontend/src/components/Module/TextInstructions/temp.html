HTML document is composed of a tree of simple HTML elements, which is a component that add semantics and formatting to parts of document.

In HTML, when you are trying to create an element, you use tags which are used in order to declare what is happening to the computer.

All the tags start off with a < symbol and end with a greater > symbol and have name of the element in the middle.

Most of HTML elements need both a opening tag and closing tag which can be differentiated by using a / in the closing tag.

For example, in this paragraph tag you can see that there is an opening tag and a closing tag and the text is placed in the centre of it.

<p> Hey </p>

HTML Tags are often nested within each other in order to separate different parts of the html document into separate section.

By looking at this example you can see that, you need to place the inside element after the start of the first tag and before the end of the tag.

<div>

 <p> Hey </p>

</div>


Think of the nesting like a tree, so the outside tag which is the is <div> is called the parent tag and the elements inside it Eg: The <p> tag)are the child tag.

Every HTML page follows this basic structure to create a webpage:

<!DOCTYPE HTML>
<html>
    <head>
        <title>Introduction</title>
    </head>
    <body>
    </body>
</html>

To start, the HTML file <!DOCTYPE HTML> tag is used. It is a declaration tag that lets the web browser know what type of document it is going to process.

The next tag is the <html> tag. This is the main tag as it contains everything related to the HTML document that will be read by the web browser. The whole HTML document is nested within the tag

As you can see, the HTML page is mainly divided into two sections: the head and the body.

The HTML <head> tag contains the information about the web page such as the title of the website, style sheets and JavaScript code. Nothing that is placed in the head tag will be rendered on the actual website, but it acts as a background for elements that will be rendered.

The <title> tag is used for the title of the website that shows up on the top of the website

The <body> tag is where the actual content of the HTML page is placed. Everything that is added to the tag will be rendered onto the screen. The order in which elements are added onto the HTML page are the way in which they appear on the website.

So by following this overall structure we are fully able to build an HTML page.

Headings are the title that you put on an HTML page in order to break up sections on a page.

A Heading Element as both an opening and a closing tag and the text in placed inside the tags. For example:

<h1> Introduction </h1>

There are six Heading Elements in HTML which all have a different size and a different level of hierarchy on the page ranging from <h1> to <h6> .

The biggest size is the <h1> tag and the smallest in size in the <h6> tag.

The heading tag that you choose depends on how big and where you want the Headings to appear on the page.

If you are creating the page title you might use <h1> but if you are writing a subsection on the page after the heading, you can use any of the smaller tags.

For paragraphs, there is only 1 type of tag known as the <p> tag.

It is used for writing paragraphs

It has an opening and a closing tag and you can enter as much text as you want within the two tags. The text in the middle is what renders on the website. For example:

<p> This is a paragraph </p>

All the text that is placed inside a paragraph tag is rendered on one line.

If the user wants to add the text on a new line they will have to use a separate <p> tag for the text. Using the separate tags allows for automatic space between the two paragraphs.

The lists tag is used to create lists on an HTML page.

There are two main types of list on HTML, the unordered list and the ordered list.

Each item within a list is defined with an <li> tag where the text is placed inside the opening and the closing tag.

Every item in the list is rendered according to the order in which they are placed in the inside the list.

Unordered list

An unordered list is defined using a <ul> tag.

For a list of fruits, an unordered list uses the following code:

<ul>
    <li>Apple</li>
    <li>Banana</li>
    <li>Mango</li>
    <li>Pear</li>
</ul>

The <ul> tag acts as a parent tag and each list item is nested within the tag.

When the list is rendered, each item in the list comes up as a bullet point.

Ordered list

An ordered list is defined using a <ol> tag.

For a list of numbers, an ordered list uses the following code:

<ol>
    <li>One</li>
    <li>Two</li>
    <li>Three</li>
    <li>Four</li>
</ol>

Similar to the unordered list, the <ol> tag acts as a parent tag and each list item is nested within the tag.

When the list is rendered, each item in the list shows with a number instead of a bullet point.

HTML also has some inbuilt elements that can be used in order to style an HTML text element such as paragraphs, headings and list items.

These tags are usually nested inside the outer text element are used to emphasise a particular part of the text.

There are multiple types of text styling tags. Here are a few of the most common examples.

Bold

Used to bold the text. Uses a <b> tag

For Example:

<p> This text is <b> bold </b> </p>

As shown in the above example, the <b> tag is nested within the paragraph tag. In the text, only the ‘bold’ part of the text gets emphasised.

Italics

Used to italicise the text. Uses a <i> tag

For Example:

<p> This text is <i> italicised </i> </p>

As shown in the above example, the <i> tag is nested within the paragraph tag. In the text, only the ‘italicised’ part of the text gets italicised.

Underline

Used to underline the text. Uses a <u> tag

For Example:

<p> This text is <u> underlined </u> </p>

As shown in the above example, the <u> tag is nested within the paragraph tag. In the text, only the ‘underlined’ part of the text gets underlined.

The <div> tag is one of the most popular HTML elements.

It is a division or container elements that helps us separate the html page into different sections. They allow us to group specific html elements together.

Each <div> can contain numerous amount of elments within it.

Divs themselves do not have some specific visual design but we are able to customise them. When some style is applied to a div, also gets applied to the HTML elements that are nested within that div tag.

For example:

<div>
    <h2>Section 1</h2>
    <p>This is the first section</p>
</div>

<div>
    <h2>Section 2</h2>
    <p>This is the second section</p>
</div>

By looking at the above examples, the different sections are split into two separate containers by using the div tag and nesting the elements within it.

For each HTML element, additional features can be added to it by using an attribute.

An attribute is content that is added to the opening tag of an HTML element and can be used in multiple ways such as styling an element or giving an element a unique value.

In order to define an HTML attribute, it has two properties, the name and the value. Each attribute is defined as name=‘value’

In HTML code an attribute looks like this:

<div id=‘numbers’>

In this case, the name of the attribute is ID and the value of the attribute is numbers.

The id attribute that is shown in this example is one of the most common attributes that is used. It is used to uniquely identify a specific element on the page.

Some attributes can work on all elements. These are known as the global attributes . For Example: Id, Class, Title.

But there are also some specific attributes that only exist for some of the HTML elements. For Example: The datetime attribute can only be used inside the datetime element

Images can also be added onto an HTML file in order to showcase something or enhance the design of the page.

These images are added using an <img> tag.

This element do not usually have a closing tag so they are written in <img /> format with the attributes inside the tag. This is referred to as a self-closing tag.

The code for an image tag looks like this:

<img src=‘../images/photo.png’ width=‘300px’ height=‘200px’/>

Each image must contain a src attribute which is the link to the image. The image that you use can be of any image format such as GIF, JPEG or PNG.

The size of the image can also be specified by using a width and height attribute.
